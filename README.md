# 數位敘事力教學工具箱

自有課堂工具箱，整合 30+ 個無 AI 原生小工具、獨立 Gems 資源區與 Chrome Web Store 系列入口。

## 開發

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## 產品原則

- 課堂工具皆在本站內操作，不導向第三方工具庫。
- 學生資料只存於瀏覽器 localStorage，可匯出 JSON 備份或清除。
- Gems 資源與 Chrome 擴充功能獨立於無 AI 工具清單。
- 介面採 Neon Circuit 調性，但優先投影可讀性、低干擾與行動裝置操作。
